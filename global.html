    (async) createCollection(req, res) → {Promise}

    Request body contains all text information, an array of object literals containing the structure of the content schema is created by using a map on the request body content array. Here the previous request where createFilesForCollection is called, the file name is stored into the content array in the client where the map than uses the file name to locate the file and create a buffer inside of the object literal. A new collection object is than created containing the array of object literals. The mongoose middleware than persists the collection document. Temp files are destroyed upon successful persistance
